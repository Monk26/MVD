GUI аля база МВД

Использованные сторонние библиотеки:
1. Tkinter для разработки графического интерфейса
2. mysql для хранения данных в СУБД


Функциональные возможности распределены для 2 типов users:
1. Функциональные возможности для граждан: регистрация и авторизация в
системе, подача заявлений и обращений с указанием: типа обращения, описания
проблемы и местоположения (адреса).

2. Функциональные возможности для сотрудников МВД: управление данными о
происшествиях, контроль за состоянием безопасности в городе, просмотр
информации о зарегистрированных обращениях и инцидентах.


Разграничение полномочий и доступа к просмотру данных будет исходя от
статуса пользователя, им будут обычные граждане и офицеры МВД, которые уже
имеют доступ и полномочия к работе с документами за счёт некой админ-панели.
Преимущества системы: автоматизация рутинных процессов обработки
обращений, централизованное хранение данных о происшествиях и
пользователях, простота использования благодаря интуитивному интерфейсу.
